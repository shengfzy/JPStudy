## 动词三分类（基本形）

- 一类动词（五段动词)
  
  う段　→ い段+ます
  
  - 结尾是う段非る的动词，肯定是5段动词
  
  - あ段+る、う段+る、お段+る
  
  - 特列
    
    - 切る
    
    - 帰る
    
    - 走る
    
    - 入る

- 二类动词
  
  去る＋ます
  
  - い段＋る
  
  - え段＋る

- 三类动词
  
  - 来る　→ 来ます
  
  - する　→ します

- 用法
  
  - 名「人」は　動「基本形」ことが　できます「表示能力，能够做～事情」
  
  - 名は　動「基本形」ことです「当谓语部分的内容是一种动作、行为时」
  
  - 動「基本形」前に、〜
  
  - 動「基本形」な　「禁止」
    
    - 用于禁止听话人做某事
  
  - 小句1（动词基本形/ない形）と，小句2　「只要～就～」
  
  - 動「基本型/ない形」ことが　あります
  
  - 動「基本形/ない形」つもりです
  
  - 動「基本形/ない形」ことに　します/しました
  
  - 動「基本形/ない形」ことに　なりました
  
  - 小句1「基本型」ために、小句2　/ 名の　ために、小句　「目的」
  
  - 動「基本形」のに「用途」「基准」
  
  - 小句１「基本型/ない形」ように、小句２
  
  - 小句「基本形/ない形」ように　なります 「表示事物自然的发展变化」
  
  - 小句「基本形/ない形」ようにします

## て形

- 一类动词（五段动词）
  
  - す　→ して
  
  - く　→ いて
  
  - ぐ　→ いで
  
  - う、つ、る　→ って
  
  - ぶ、む、ぬ　→ んで
  
  特例：行く　→ 行って

- 二类动词
  
  - る　→ て

- 三类动词
  
  - 来る　→ 来て
  
  - する　→ して

- 用法
  
  - 并列
    
    - 動て　動「相继发生」
    
    - 動てから　動「相继发生」
    
    - 一类形く　て　一类形「并列」
    
    - 二类形で　二类形「并列」
    
    - 名１で　名２「并列」
  
  - 動て　ください「请求某人做某事」
  
  - 動て　います「正在进行」「结果状态」「反复、习惯」
  
  - 動ても　いいです「表示许可」
  
  - 動ては　いけません「表示禁止」
  
  - 授受关系
    
    - 動て　あげます「为别人做某事」
    
    - 動て　もらいます「请别人做某事」
    
    - 動て　くれます「别人为我做某事」
  
  - 〜てほしい「表示对他人的愿望」
  
  - 补助动词
    
    - 動て　しまいます
    
    - 他動て　あります「暗示有人做某事」
    
    - 動て　おきます「实现做某事」
    
    - 動て　みます「尝试做某事」
  
  - 小句1ても、小句「即使～」
  
  - 小句1て、小句2「原因、理由」
  
  - 動「て形」ばかりいます

## 未然形（ない形）

- 一类动词（五段动词）
  
  - う段　→ あ段
  
  - 特例：う　→ わ

- 二类动词
  
  - 去掉る

- 三类动词
  
  - 来る　→ 来ない
  
  - する　→ し、せ、さ　ない

- 用法
  
  - 動「未然形」ないで　ください「请不要做～」
  
  - 動「未然形」なければ　なりません「必须做～」
  
  - 動「未然形」ないと　いけません「必须做～」
  
  - 動なくても　いいです 「不做～也可以」
  
  - 動「未然形」ない　ほうが　いいです「不做～更好」
  
  - 小句1（动词基本形/ない形）と，小句2　「只要～就～」
  
  - 動「基本型/ない形」ことが 　あります
    
    - 表示有时会发生某种事态。往往和 たまに　時々 等副词呼应使用
  
  - 動「基本形/ない形」つもりです
  
  - 動「基本形/ない形」ことに　します/しました
  
  - 動「基本形/ない形」ことに　なりました
  
  - 小句１「基本型/ない形」ように、小句２
  
  - 小句「基本形/ない形」ように　なります 「表示事物自然的发展变化」
  
  - 小句「基本形/ない形」ようにします

## た形

- 一类动词（五段动词）
  
  - す　→ して
  
  - く　→ いて
  
  - ぐ　→ いで
  
  - う、つ、る　→ って
  
  - ぶ、む、ぬ　→ んで

特例：行く　→ 行って

- 二类动词
  
  - る　→ て

- 三类动词
  
  - 来る　→ 来て
  
  - する　→ して

- 用法
  
  - 動「た形」ことが　あります「曾经～过」
  
  - 動「た形」後で、〜
  
  - 動「た形」ほうが　いいです「做～更好」
    
    - 動「ない形」ほうが　いいです「不做～更好」
  
  - 小句「た形」ら、小句
    
    - 否定用小句「未然形」なかった

## 条件表达方式

- 〜と
  
  - 用于前项一旦成立，后项就必然成为现实的复句里
  
  - 不用于意志、请求
  
  - 用于过去式，表示发现了某种情况
    
    - 窓をあけると、富士山が見えました

- たら
  
  - 万能
  
  - 用于过去式，表示发现了某种情况
    
    - 窓を開けたら、富士山が見えました

- ば
  
  - 不用于意志和请求
  
  - 不用于过去式

- なら
  
  - 用于对话，多用于以对方的某种意志或愿望为前提而提出某种建议
    
    - 新鮮な魚を買いたいんですが。　ー魚を買うなら、駅前のスーパーがいいですよ

## 比较级和最高级

- 名１は　名２より　一类形/二类形です

- 名１より　名２のほうが　一类形/二类形です

- 名１は　名２　ほど　一类形くないです/二类形では　ありません

- 名１の　中で　名２が　一番　一类形/二类形です
  
  - 如“名1”表示场所或时间，则后面不加の中，而使用 名１「場所/時間」

- 一类形/二类形(な)　ほうが　いいです

- 動「た形」ほうが　いいです
  
  - 本句型用于在两种事物中进行选择时，句尾多加よ

- 動「ない」ほうが　いいです

## 格助词的用法

- は   
  
  - 提示主题
  
  - 比较
  
  - 主题是旧信息

- に
  
  - 具体的时间后
  
  - 地点+あります/います
  
  - 友達に　合います(动作的对象)
  
  - 表示行动的目的
  
  - 車に　乗ります
  
  - 二类形/名 に　なります/します
  
  - 名に/動「基本型」「用途」「基准」标准，用作比较的对象

- で
  
  - 会议、展览 + あります
  
  - 交通工具
  
  - 动作的地点
  
  - 动作的手段、原材料
  
  - 名で 表示原因或理由

- を
  
  - 他动词的对象
  
  - 经过、离开

- か
  
  - 或者，对若干个事物之间的选择
  - 疑问词+か　表某时、某地、某人等

- も
  
  - 全盘否定
  - 也

- と
  
  - 和，表并列
  - 只要...就...

- や
  
  - 全局中的列举

- が
  
  - ～が　あります/います
  - 情感的对象：好き　嫌い　怖い　いいです
  - 有关能力的词：分かります　できます　上手です　下手です　苦いです
  - 名１の中で　名２が　一番　一类形/二类形です
  - 名１は　名２が　欲しいです
  - 〜が　動たいです
  - 自动词的主语
  - 叙述直接观察到的事物
  - 主语是新信息
  - 疑问词做主语
  - 表铺垫或转折
  - 修饰名词的动词小句的主语
