### 第３課

- ここ/そこ/あそこは　名です

- 名は　どこですか

- 名も　名です
  
  - 助词も相当于汉语的也

- 名は　名ですか　名ですか
  
  - 答案有多种，询问其中的一种

-  名は　いくらですか
  
  - 询问价钱

- こちら/そちら/あちら/どちら
  
  - ここ/そこ/あそこ/どこ的礼貌说法

- ですか　「确认」

### 第４課（存在句）

- 名「場所」に　名「物/人」が　あります/います

- どこに　ありますか/いますか

- 名と　名　「并列」

- 疑问词 + も+动「否定」
  
  - 表示全面否定

### 第５課（时间）

- 今　〜時　〜分です

- 动ます/动ません/动ました/动ませんでした

- 名「時間」に　动
  
  - 表示动作发生的时间时，要在具体时间词语后面加上助词に
  
  - 今　昨日　今日　明日　毎日　去年　来年等词后不能加に
  
  - 星期后一般加に，但也可以不加

- 名「時間」から　名「時間」まで　动

- いつ　動ますか
  
  - 询问某动作或事态进行的时间
  
  - 询问的时间很具体时，在表示时间的词语后面加に（何時に　何曜日に　何日に）
  
  - 询问持续性动作或事态的起点或终点时，用いつから　何曜日まで等

- は「对比」

- 毎朝　何時に　起きますか
  
  - 当两个名词都表示时间，而前面的名词中包含“每”时，两个名词之间不加の。当前面的名词中不包含“每”时，可以加の，也可以不加の

- 〜ごろ
  
  - 接在表示时间的词语后面（分　時　日　曜日　月　年），表示大致的时间范围

### 第７課（动词三分类）

- 名を　動

  - 动作发生的场所

- 名か　名
  
  - 对若干个名词进行选择时

  - 花钱购物或者不花钱的索取，请把～给我的意思

- 何的读法
  
  - 读なん的情况：何の　何時　何分　何曜日

- そうですか
  
  - 读降调时，表示理解了所听到的新信息。相当于汉语的是吗

- そうですね
  
  - 用于同意对方的提议

- じゃあ
  
  - 结果别人的话题来发表自己的看法，或中断谈话时使用，相当于汉语的那么
  - 正式场合用では

- すみません「搭话」

### 第９課（一类形容词）

- 一类形容词
  
  - い结尾
  
  - 汉字+い（熱い　冷たい　寒い　暖かい）
  
  - 特殊：綺麗（きれい）　詳細（しょうさい）　愉快（ゆかい）　嫌い（きらい）

- 变形
  
  - 肯定：い＋です
  
  - 否定：い　ー　く　ないです　/ 　く　ありません
  
  - 过去式：い　ー　かった
  
  - 过去否定：い　ー　く　なかった　/ 　く　ありませんでした

- 名は　一类形です

- 一类形 + 名

- を　→   は
  
  - は表示对比时，如果前面有を，需要去掉

- 表示程度的副词
  
  - とても/たいへん　＞　少し/ちょっと　＞　あまり〜ません　＞　全然〜ません

- 多い　少ない不能直接修饰名词，需要说たくさんの本

- 熱くないですか
  
  - はい、熱くないです
  
  - いいえ、熱くないです

- ちょうど　いい
  
  - 表示时间或者分量等合适

### 第１０課（二类形容词）

- 名は　二类形です/でした

- 名は　二类形では　ありません　/ では　ありませんでした

- 名は　二类形な　名です

- 名でした
  
  - 名词做谓语时的过去形式。其否定形式是 名は　では（じゃ）ありませんでした

- どんな　名
  
  - 询问人或者事物的性质时，相当于汉语的什么样的

- 何の　どんな
  
  - 何の：询问内容和材料
  
  - どんな：询问性质

- でも　そして
  
  - でも：表转折关系的连词，一般用于口语。正式用しかし
  
  - そして：表并列关系的连词

- 一昨日は　雨でした
  
  - 表示自然现象的动词句，常使用雨です　雪です等名词句的形式

- ところで
  
  - 转换话题时使用

- 「場所」中
  
  - 表示场所范围内的全部
  
  - 世界中　日本中　学校中　一日中　一晩中　一年中

- 形容词+の
  
  - 代指前面提到的事物

### 第１１課

- 名１は　名２が　一类形/二类形です
  
  - 用于 好きです　嫌いです　怖いです等表示情感的形容词

- 名１は　名２が　分かります/できます
  
  - 用于 分かります　できます　上手です　下手です　苦手ます等有关能力的词语

- 名や　名
  
  - 只列举许多项目中的两项
  
  - 列举所有内容用と

- 〜から　だから
  
  - 一类形 / 動：用から
  
  - 二类形 / 名：用だから　ですから　（ですから是だから比较礼貌的说法）

- 表示频率的副词
  
  - いつも　＞　よく　＞　時々　＞　たまに　＞　あまり〜ません　＞　全然〜ません

- ホテルで　写真展が　あります
  
  - 写真展　パティー　会議等表示事件的名词，表示在某处发生某事。某事发生的场所用で表示

- いかがですか
  
  - いかがですか是どうですか的礼貌表达方式
  
  - 肯定回答：どうも　ありがとうございます
  
  - 否定回答：結構です

- そうですね
  
  - 表示同意对方提议
  
  - 表示说话人在思考接下来要说的内容

- 迷います
  
  - 用于难以立即作出选择的场合
  
  - 表示迷路

- 気に入りました
  
  - 用于第一次见到某事物而感到中意

### 第12課（比较级、最高级）

- 名１は　名２より　一类形/二类形です

- 名１より　名２の　ほうが　一类形/二类形です

- 名１は　名２ほど　一类形く　ないです/二类形では　ありません

- 名１の　中で　名２が　一番　一类形/二类形です
  
  - 如“名1”表示场所或时间，则后面不加の中，而使用 名１「場所/時間」

- 名１と　名２と　どちらが　一类形/二类形ですか
  
  - 两个事物比较提问

- どの　名/いつ/どれ/だれ/何が　一番　一类形/二类形ですか
  
  - 三个以上事物比较的提问

- 最近
  
  - 表示从较近的过去到现在为止的一段时间，所指时间长短因人而异

- 〜が　いいです
  
  - 表示选择，选哪个比较好（お土産は　何が　いいですか）
  
  - 我更喜欢～（私は　焼酎より　ワインの　ほうが　いいです）

- やっぱり
  
  - やっぱり是やはり较随便的说法，用于日常谈话。表示某信息或事态的发展与自己的预测一致，意思是“正如自己预料的那样”

- 〜が・・・。
  
  - 在遇到难以启齿的事，或者拒绝别人的请求时经常这样做

### 第13課（量词）

- 名「数量」+ 動

- 名「時間」＋動
  
  - 表示动作、状态的持续时间。这个时候表示时间数量的词语后面不能加に（李さんは　毎日　7時間　働きます）

- 名「時間」に　名「次数」＋動
  
  - 表示在一定时间内进行若干次动作时，使用这个句型 （李さんは　一週間に　2回　プールへ　行きます）

- 名「場所」へ　動に　行きます/来ます
  
  - 表示移动行为的目的时使用，动词用ます形去掉ます的形式

- 名「数量」で
  
  - 用于不称重量而以数个的方式售物
  - 数量是1个时，不加で

- 週に　2回
  
  - 完整表达是“一週間に　2回”
  - 一日に　→ 日に
  - 一か月に　→ 月に
  - 一年に　→ 年に

- 生ビールを　３つ

- くらい/ぐらい
  
  - 加在表示数量的词后面，表示大概的数量
  - 常与副词だいたい同时使用
  - 询问时使用 どのぐらい

- どのぐらい　かかりますか
  
  - 用于时间和钱，相当于汉语的“需要、花费”

- とりあえず
  
  - 从具有可能性的几种动作或事项中，暂且选择一种先做，暗示其他动作或事项等一会儿还会随后进行，在饭店点菜时经常使用

### 第15課（动词て形）

  - 表示禁止时

- 無理を　しては　いけませんよ

- ゆっくり
  
  - 好好地，慢慢地

### 第17課

- 名１は　名２が　欲しいです
  
  - 表示愿望

- 名１は　名２を　動たいです
  
  - 表示想要～
  
  - 陈述句的主语是私、疑问句的主语是あなた
  
  - たい的对象可以用を也可以用が、但是欲しい的对象只能用が

- 全面否定も的用法
  
  - 名＋を＋動　→ 疑问词も+动词的否定形式
  
  - 名＋に、から、と＋動　→ にも、からも、とも＋動
  
  - 名＋へ　動　→ へも、も　＋　動

- 動　ませんか「有礼貌的提议」

- 動　ましょう「偏口语的提议」

- 疑问词+でも
  
  - 表示任何情况下失态都相同
  
  - 何でも　いいです

- 今年中に
  
  - 表示该段时间结束之前的意思
  
  - 今日に　明日に　今年中に：读じゅうに
  
  - 〜週中に　〜月中に：读ちゅうに

- ぜひ
  
  - 一定，必定，后续〜たい　〜てください　〜ましょう等，是表达强烈愿望，意志，提议的形式

### 第19課（ない形）

### 第20課（基本形）

- 名「人」は　動「基本形」ことが　できます
  
  - 表示能力，能够做～事情
    
    - 森さんは　中華料理を　作る　ことが　でいますか
    
    - はい、できます

- 名は　動「基本形」ことです
  
  - 当谓语部分的内容是一种动作、行为时使用这个句型
    
    - 私の趣味は　切手を　集める　ことです
    
    - 小野さんは　車を　運転する　ことが　できません
    
    - 小野さんは　車の　運転が　できません

- 動「基本形」前に、〜
  
  - 表示一个动作在另一个动作之前发生时，使用这个句型

- 疑问词+か
  
  - 疑问词“いつ　どこ　だれ”等词后续”か”、表示“某时、某地、某人”等意思，用于陈述句
  
  - 何度か（几次）

- 写真を　撮る　ことが　できます「可能或允许」

- みんなで
  
  - で表示动作主体的样态，如一人で　自分で　全員で
    
    - みんなで　餃子を　作ります

- 趣味　興味

- ごちそします
  
  - 请别人吃饭

### 第21課（た形）

- 動「た形」ことが　あります
  
  - 表示过去的经历。大致相当于汉语的（曾经）～过。用于至少半年以前发生的事
  
  - 否定形式：（一度も）動「た形」ことが　ありません
  
  - 疑问形式：動「た形」ことが　ありますか
    
    - はい、あります
    
    - いいえ、ありません

- 動「た形」後で、〜
  
  - 表示一个动作在另一个动作之后发生时
    
    - 会社が　終わった　あとで、飲みに　行きます
    
    - 仕事の後で、映画を　見ます

- 動「た形」ほうが　いいです
  
  - 本句型用于在两种事物中进行选择时，句尾多加よ
  
  - 否定句形：動「ない」＋ほうが　いいです

- 動ましょうか「提议」
  
  - 用于提议对方和自己一起做某事或向对方提议自己为对方做某事

- その　ほうが　いいです「接受对方的意见」
  
  - これ/それ/あれ　の　ほうが　いいです
  
  - この/その/あの　ほうが　いいです

- 九時過ぎ
  
  - 〜過ぎ用在表示时间或年龄的词语后面表示超过了该时间或年龄
  
  - 时间：数词+時/分＋過ぎ
  
  - 年龄：数量+過ぎ

### 第22課（动词简体形）

- 小句けど、小句「转折」「铺垫」
- 简体疑问
  - これ、日本製？
- うん　ううん
  - はい　→ うん
  - いいえ　→ ううん
- 简体助词
  - かな：本来只用于自言自语的场合。如有听话人在场时则表示通过让对方听到自己的自问自答而向对方提供一种不太确实的信息

## 语气助词

- よ
  
  - 用于提醒对方注意其不知道的信息，表告知、提醒、轻微的警告

- ね
  
  - 当确定自己和对方的意见相同时，表征求意见

- よね
  
  - 当对自己的意见是否和对方相同没有足够的把握，表征求意见

- かな
  
  - 用于简体形，表自言自语或让对方听到提供一种不太确实的信息

- でしたね
  
  - 从前知道但记不太确切的事情进行确认

- でしょう
  
  - 表推测
  
  - 表 ...是这样吗？升调
  
  - 表叮问，或意见不一致
